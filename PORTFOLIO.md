## 🌐 For Portfolio Website (Project Showcase)

### 🚀 Highlight: AI Code Review Agent

**The Problem:**
Manual code reviews are a massive bottleneck in agile environments. Reviewers suffer from context fatigue, leading to missed vulnerabilities and superficial feedback.

**The Solution:**
I built the *AI Code Review Agent* — a unified platform that acts as a first-pass reviewer on pull requests. By routing code diffs through advanced LLMs, it provides immediate feedback on structure, efficiency, and security.

**Tech Stack deeply explored:**
- **Frontend:** React 18, Vite, Tailwind CSS, Context API.
- **Backend:** Python 3.11, FastAPI, Pydantic, async DB drivers.
- **Database:** PostgreSQL, SQLAlchemy ORM, Alembic.
- **DevOps:** Docker, Nginx reverse-proxies, Environment Management.
- **AI/ML:** OpenRouter API abstraction for agnostic LLM routing.

**Impact:**
- Simulated 40% reduction in code-review turnaround times.
- Prevented potential cross-site scripting (XSS) and SQL injection vulnerabilities during testing phases.
- Delivered a sleek, modern UI with a custom glassmorphism design system that elevates the user experience.
